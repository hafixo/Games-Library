# Games-Library

The app bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app). 

## Installation

Download or clone this repo, then run the following command:

```sh
npm install
```

## Usage example

You can run the app by using npm script:

```sh
npm start
```
